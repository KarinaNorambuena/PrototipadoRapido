Ayuda Para Girar Llaves

Este instrumento de ayuda está diseñado para personas con restricción de movimiento en las manos, que presenten dificultad para pinzar o hacer movimientos finos (como la artrosis, osteoartritis, o tendinitis) , específicamente para ayudar a girar la llave y poder entrar a casa con mayor facilidad.

Su forma es ancha y gruesa y sus costados son redondeados en forma de ondas, que hace más fácil el agarre sin la necesidad de utilizar la fuerza, como no es necesario pinzar las llaves, la muñeca posee mayor estabilidad, lo que prevé dolencias en toda la extremidad motora.

También se pensó en una adaptabilidad un poco más amplia, añadiendo una sección que permite ser mordido para usuarios extremos que no podrían hacer uso de sus manos. Se añadieron dos palancas independientes para poder rotar cada llave cuando se necesite.

El objeto tiene un largo máximo de 10,5cms, ancho de 5,5cms y 3cms de grosor en la zona más ancha.

Modelo Fabricado

Vista 1

El modelo era muy delgado, corto y pequeño, de todas maneras debíamos ejercer mucha presión para sostenerlo bien y era incómodo. A pesar de tener espacio para dos llaves, al estar enganchadas de un mismo eje se rotaban las dos al mismo tiempo.

Vista 2

También existía el problema de que la tuerca no se sujetaba bien con su soporte, se salía el soporte y costaba mucho rodarlo para hacer salir la llave y eso, además de ser pequeño, no iba a facilitar la acción a personas con dificultades en las manos.


Propuestas de Rediseño

Propuesta 1

Modelo rectangular con bordes curvos, ancho y largo, simulaba un mango de bicicleta. Soportaba una sola llave, se pensó agregar silicona/goma en los laterales para mejorar el agarre. Contaba con una palanca gruesa para rotar las llaves. Es la primera propuesta con el espacio para la mordida.

Propuesta 2

El modelo 2 tenía una forma más orgánica, también soportaba una sola llave e igualmente se pensó agregar goma/silicona a los costados para mejorar el agarre. La salida de las llaves emulaba la extracción de las hojas de un cuchillo cartonero.

Propuesta 3

Esta propuesta es lo más cercano al modelo final, realizamos dos formas, la forma de papa y una más curva y nos decidimos por la segunda ya que permitía un mejor agarre debido a las mismas curvas. Le agregamos el espacio para la mordida y una palanca para girar y extraer la llave.

Conceptos Centrales

Estabilidad, músculo motor, gesto técnico, prevención, movilidad, comodidad, facilidad, ergonómico.



Conclusiones Finales

Observamos y analizamos a través del prototipo extraído de la web, aquellas características prácticas que resolvían las dificultades principales del usuario y fuimos mejorando las propiedades poco funcionales, como el tamaño, la forma, la empuñadura, (buscando la manera de hacerlas más cómodas y mejorar el agarre) el sistema para extraer las llaves (queríamos utilizar un sistema que permitiera la incorporación de 2 llaves con ejes independientes y con una palanca o sistema fácil de rotar), y el uso. El objeto esta rediseñado para lograr utilizarlo con las manos y también con la boca para usuarios extremos.
